Flask project

Bienvenue sur mon site de recherche de caractères unicodes ou vous pourrez recherchez des caractères unicodes en fonction de différents critères . Celui-ci est réalisé en Flask qui est un framework de python avec bootstrap pour le visuel qui est un framework CSS.

Voici les différentes informations d'un caractère unicode :
  • Son caractère
  • Son nom
  • Son point de code en hexadécimal
  • Son point de code en décimal
  • Sa catégorie

Voici quelques exemples de recherches possibles unicodes :

  • Le caractère unicode n°55
  • Tous les caractères avec un coeur dans leur nom
  • Le caractère unicode Ż
  • Le caractère unicode de point de code hexa U+1f354
Pour faire une recherche il faut vous rendre vers la page de recherche en cliquant sur le boutton situé juste en dessous
